Documentation de l'application Librairie ABC
Installation
Cloner le dépôt
git clone https://github.com/votre-repo/librairie-abc.git
cd librairie-abc


Installer les dépendances
pip install -r requirements.txt


Pour exécuter l'application
Exécutez le script run.sh : bash chmod +x run.sh ./run.sh

Tests
Pour exécuter les tests, utilisez pytest : bash pytest test_librairie_abc.py
